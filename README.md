# Todo Web App

A full-stack Laravel TODO web application with MySQL backend and React frontend. Features task creation, updating, and deletion with a modern, responsive UI. Dockerized for easy setup and development.

## Features

- Create, update, and delete tasks
- Responsive frontend built with React
- Backend powered by Laravel and MySQL
- Dockerized environment for easy setup
- Ready for local development and testing

## Prerequisites

- [Docker Desktop](https://www.docker.com/products/docker-desktop) (with WSL2 enabled)
- [Git](https://git-scm.com/)
